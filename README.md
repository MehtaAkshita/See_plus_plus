# See_plus_plus
An application for blind (differently abled) people to read the text and symbols from road sign boards written in English language (min_req: voice enabled system to speak the text) (dataset)

### Link to weight file
https://drive.google.com/drive/folders/1gozqF319ni9BCkE3v-PEopBK3JZ4cBaN?usp=sharing

### How to run it
python runMain.py --east frozenEaseTextDet.pb --image images/example_01.jpg
python runMain.py --east frozenEaseTextDet.pb --image images/example_04.jpg --padding 0.05
